# Altura Estates App

**Your Gateway to Premium Homes**

A comprehensive full-stack real estate application for browsing, comparing, and purchasing premium properties across premium locations.

## 🏗️ Project Structure

```
altura-estates-app/
├── backend/              # Node.js/Express API Server
├── web/                  # Next.js Web Application
├── mobile/               # React Native Mobile App
└── shared/               # Shared Types & Constants
```

## ✨ Features

### User App Features
- 🔍 Property Search & Filters
- 🆕 New Launches
- 🏰 Luxury Villas
- 🏢 Apartments
- 🏪 Commercial Spaces
- ⚖️ Property Comparison
- 💰 Home Loan EMI Calculator
- 📅 Schedule Site Visit
- 💬 WhatsApp Inquiry
- 🤖 AI Property Recommendations
- ❤️ Saved Properties

### Areas Covered
- Kokapet, Tellapur, Narsingi, Kollur
- Rajendra Nagar, Gachibowli, Financial District

## 🛠️ Tech Stack

- **Web**: Next.js 14, React, TypeScript, Tailwind CSS
- **Mobile**: React Native, Expo
- **Backend**: Node.js, Express, TypeScript
- **Database**: MongoDB
- **Auth**: JWT + Firebase
- **DevOps**: Docker, GitHub Actions

## 📋 Prerequisites

- Node.js v18+
- MongoDB 6.0+
- Docker & Docker Compose

## 🚀 Quick Start

### Using Docker
```bash
docker-compose up -d
```

### Manual Setup
```bash
# Backend
cd backend && npm install && npm run dev

# Web (new terminal)
cd web && npm install && npm run dev

# Mobile (new terminal)
cd mobile && npm install && npm start
```
